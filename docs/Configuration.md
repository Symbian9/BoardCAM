# Configuration

滑雪板参数(单位: mm)
* Overall Length: 总长度
* Running Length: 行程长度
* Nose Width: 板头宽度
* Nose Length: 板头长度

* Tail Width: 板尾宽度
* Tail Length: 板尾长度

* Waist Width: 板腰宽度
* Sidecut Radius: 侧切半径(此参数越小, 转弯半径越小)
* Sidecut Depth: 侧切深度(侧切半径越小, 侧切深度越大)

* Stand Setback: 站位偏移
* Stand Width: 建议间距(两脚之间的站距)
* horizontal spacing: 相邻嵌件之间的水平间距
* vertical spacing: 相邻嵌件之间的垂直间距
